# Reinforcement Learning with Imagined Goals (RLIG)

<br>

## Initial Setup 

```console
./setup.sh
```

<br>

## Install RLIG locally through pip

```console
# safe install
./pip_install.sh 

# manual install
pip install .
```

<br>

## Editing RLIG

Upon changing any files located in src/rlig, re-execute:

```console
./pip_install.sh 
```
